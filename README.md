# My own printf

## General information


This is a `print` command that has been written from scratch using the **NASM** assembly language

My command supports the specifiers:

+ `%d` - decimal number
+ `%o` - octal number
+ `%u` - unsigned number
+ `%b` - binary
+ `%x` - hexadadecimal
+ `%s` - string
+ `%c` - char

## Program start

For Linux

```c
make
```
